# [Sum of Binary Tree](https://www.geeksforgeeks.org/problems/sum-of-binary-tree/1)
## Easy
Given a binary tree, find the sum of values of all the nodes.&nbsp;
Examples:
Input: root = [15, 10, 20, 8, 12, 16, 25]Output: 106Explanation: The sum of all the nodes is 15 + 10 + 20 + 8 + 12 + 16 + 25 = 106.
Input: root = [1, 3, 2]Output: 6Explanation: The sum of all the nodes is 1 + 2 + 3 = 6.
Input: root = [1, 2, N, 4, N]Output: 7Explanation: The sum of all the nodes is 1 + 2 + 4 = 7.
