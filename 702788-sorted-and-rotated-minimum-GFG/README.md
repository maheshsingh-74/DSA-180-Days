# [Sorted and Rotated Minimum](https://www.geeksforgeeks.org/problems/minimum-element-in-a-sorted-and-rotated-array3611/1)
## Easy
A sorted array&nbsp;of distinct elements arr[]&nbsp;is rotated at some unknown point, the task is to find the minimum element in it.&nbsp;
Examples:
Input: arr[] = [5, 6, 1, 2, 3, 4]
Output: 1
Explanation: 1 is the minimum element in the array.
Input: arr[] = [3, 1, 2]
Output: 1
Explanation: Here 1 is the minimum element.
Input: arr[] = [4, 2, 3]
Output: 2
Explanation: Here 2 is the minimum element.
Constraints:1 ≤ arr.size() ≤ 1061 ≤ arr[i] ≤ 109
